# RAG Chat Application

A Retrieval-Augmented Generation (RAG) chat application with a modern UI, document upload capabilities, and conversational AI powered by Groq.

## Features

- Chat Interface - Modern, responsive chat UI
- Document Upload - Upload PDFs and documents for RAG
- URL Integration - Add website URLs as knowledge sources
- RAG Architecture - Combines retrieval with language models
- Modern Design - Clean, Deepseek-inspired UI
- Table Support - Renders tables beautifully
- Email Sharing - Share chat responses via email
- Copy to Clipboard - Easy response copying

## Tech Stack

### Frontend
- React.js
- CSS3 (Flexbox, Grid)
- React Markdown

### Backend
- Python
- Groq API
- Vector Database (Chroma/FAISS)
- LangChain (for RAG)

## Installation

### Prerequisites
- Node.js (v14+)
- Python (v3.8+)
- Groq API Key
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/rag-chat-app.git
cd rag-chat-app

# Install dependencies
npm install

# Start development server
npm start

```
